<size=16>AnyPortrait Update Log v1.0.3</size>
(Build April 14, 2018)


<size=13>Improvements</size>

- Significant improvements in <color=blue>Screen Capture</color>
    Transparent color can be specified as background color (Except GIF animation)
    Added ability to save Sprite Sheet
    Screen capture Dialog is deleted and moved to the right screen to improve
    Support screen capture on Mac OSX
    <color=red>(Check out the enhanced screen capture features on the homepage)</color>

- Improved <color=blue>Physics Effects</color>
    Corrected incorrectly calculated inertia when moving from outside
    Some changes to preset settings as physical formulas change
    <color=red>(Physical properties changed. Please check the homepage!)</color>

- Modify the gizmo to be inverted if the scale of the object is negative
- When replacing the texture of the mesh, Script Functions that can be replaced with an image registered in AnyPortrait has been added



<size=13>Fixed issues</size>

- Fixed an issue that caused data errors to occur when undoing after creating or deleting objects
- Fixed a problem that when importing animation pose, data is missing while generating timeline automatically
- Fixed an issue where other vertices were selected when using the FFD tool
- Fixed an issue where vertex positions would be strange when undoing when using FFD tool
- Fixed an issue where the modifier did not recognize that the mesh was deleted, resulting in error code output
- Fixed an issue where the clipping mesh would not render properly if the Important option was turned off
- Fixed an issue where sub-mesh groups could not generate clipping meshes
- Fixed a problem where deleted mesh and mesh groups appeared as GameObjects
- Fixed a problem where the script does not change the texture of the mesh

<size=13>수정된 문제점</size>

- 애니메이션 포즈를 Import할 때, 자동으로 타임라인을 생성하면서 데이터가 누락되는 문제
- FFD 툴을 사용할 때, 다른 버텍스가 선택되는 문제
- FFD 툴을 사용할 때 실행 취소를 하면 위치가 이상해지는 문제
- 메시 삭제후 모디파이어에서 이를 인식하지 못하여 발생하는 에러 코드
- Important 옵션이 꺼지면 클리핑 메시가 제대로 렌더링하지 못하는 문제
- 하위 메시 그룹에서 클리핑 메시를 생성할 수 없는 문제
- 삭제한 메시나 메시 그룹이 GameObject로 등장하는 문제
- 스크립트로 메시의 텍스쳐가 바뀌지 않는 문제

<size=13>Naprawiono problem</size>

- Naprawiono błąd, który powodował błędy danych podczas cofania po utworzeniu lub usunięciu obiektów
- Naprawiono problem polegający na tym, że podczas importowania animacji nie ma danych podczas automatycznego generowania osi czasu
- Naprawiono błąd, w którym inne wierzchołki zostały wybrane podczas używania narzędzia FFD
- Naprawiono problem, w którym pozycje wierzchołków byłyby dziwne po cofnięciu podczas używania narzędzia FFD
- Usunięto problem polegający na tym, że modyfikator nie rozpoznał usunięcia siatki, co spowodowało wygenerowanie kodu błędu
- Naprawiono problem, w którym przycinanie siatki nie wyświetlałoby się prawidłowo, gdyby ważna opcja była wyłączona
- Naprawiono błąd, w wyniku którego grupy podsieci nie mogły generować siatki obcinania
- Naprawiono problem polegający na tym, że usunięte siatki i grupy siatki pojawiły się jako GameObjects
- Naprawiono problem polegający na tym, że skrypt nie zmienia tekstury siatki
